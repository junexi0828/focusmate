## 시작하기

FocusMate는 팀 기반 집중 학습 및 협업 플랫폼입니다. 이 가이드는 처음 사용하는 사용자를 위한 단계별 안내서입니다.

### 필수 요구사항

- **웹 브라우저**: Chrome, Firefox, Safari, Edge (최신 버전)
- **인터넷 연결**: 안정적인 인터넷 연결
- **계정**: 이메일 주소로 회원가입

---

## 설치 가이드

### 1. 회원가입

1. FocusMate 웹사이트 접속
2. "회원가입" 버튼 클릭
3. 이메일, 사용자명, 비밀번호 입력
4. 이메일 인증 완료

### 2. 로그인

1. 이메일과 비밀번호 입력
2. "로그인" 버튼 클릭
3. 대시보드로 자동 이동

## 주요 기능 사용법

### 📊 대시보드

**목표 설정**:
1. 대시보드에서 "목표 설정" 클릭
2. 일일 목표 시간 입력 (예: 120분)
3. 주간 세션 목표 입력 (예: 10회)
4. "저장" 버튼 클릭

**세션 기록**:
1. "세션 추가" 버튼 클릭
2. 방 이름, 시간, 완료 시간 입력
3. "저장" 버튼 클릭
4. 통계에 자동 반영

**통계 확인**:
- 일일/주간/월간 통계 확인
- 목표 달성률 확인
- 세션 기록 조회

---

### 💬 채팅

**채팅방 입장**:
1. 사이드바에서 "채팅" 클릭
2. 원하는 채팅방 선택
3. 메시지 입력 후 전송

**읽지 않은 메시지**:
- 사이드바에 읽지 않은 메시지 수 표시
- 30초마다 자동 업데이트
- 채팅방 입장 시 자동으로 읽음 처리

---

### 👥 커뮤니티

**게시글 작성**:
1. "커뮤니티" 메뉴 클릭
2. "새 게시글" 버튼 클릭
3. 제목, 내용, 카테고리 입력
4. "작성" 버튼 클릭

**게시글 검색**:
1. 검색창에 키워드 입력
2. 제목과 내용에서 자동 검색
3. 카테고리 필터 적용 가능

**좋아요 & 댓글**:
- 게시글 좋아요: ❤️ 아이콘 클릭
- 댓글 작성: 댓글 입력창에 내용 입력
- 대댓글: 댓글의 "답글" 버튼 클릭

---

### 🏆 랭킹 & 팀

**팀 생성**:
1. "랭킹" 메뉴 클릭
2. "팀 생성" 버튼 클릭
3. 팀 이름, 설명 입력
4. "생성" 버튼 클릭

**팀원 초대**:
1. 내 팀 페이지에서 "초대" 버튼 클릭
2. 초대할 사용자 이메일 입력
3. "초대 보내기" 클릭
4. 상대방이 초대 수락 시 팀원 추가

**팀 멤버 확인**:
1. 팀 페이지에서 "멤버" 탭 클릭
2. 멤버 목록, 역할, 통계 확인
3. 리더는 멤버 관리 가능

**초대 관리**:
1. "내 초대" 메뉴 클릭
2. 대기 중인 초대 확인
3. "수락" 또는 "거절" 버튼 클릭

---

### 🎓 학생 인증

**인증 요청**:
1. "랭킹" > "인증" 메뉴 클릭
2. 학교 이름 입력
3. 학번 입력 (선택)
4. 학생증 사진 업로드
5. "제출" 버튼 클릭

**인증 상태 확인**:
- 대기 중: 관리자 검토 대기
- 승인됨: 인증 완료, 이메일 알림 발송
- 거절됨: 사유와 함께 이메일 발송

**보안**:
- 업로드된 파일은 암호화되어 저장
- 학번은 암호화 처리
- 관리자만 열람 가능

---

### 🎮 미니게임

**게임 플레이**:
1. "미니게임" 메뉴 클릭
2. 원하는 게임 선택
   - 🦖 Dino Jump
   - 🟡 Dot Collector
   - 🐍 Snake
3. "Play Now" 버튼 클릭
4. 게임 종료 시 점수 자동 제출

**리더보드**:
- 게임별 상위 10명 표시
- 실시간 순위 업데이트
- 팀 경쟁 점수 확인

---

### 🏅 업적

**업적 확인**:
1. "업적" 메뉴 클릭
2. 잠금/해제된 업적 확인
3. 달성 조건 확인

**업적 종류**:
- 총 세션 수
- 총 집중 시간
- 연속 출석 일수
- 커뮤니티 게시글 수

## FAQ

### Q1. 비밀번호를 잊어버렸어요
A: 로그인 페이지에서 "비밀번호 찾기"를 클릭하고 이메일을 입력하세요. 재설정 링크가 발송됩니다.

### Q2. 팀을 탈퇴하고 싶어요
A: 팀 페이지에서 "팀 탈퇴" 버튼을 클릭하세요. 리더는 탈퇴 전 리더 권한을 다른 멤버에게 양도해야 합니다.

### Q3. 읽지 않은 메시지 수가 업데이트되지 않아요
A: 페이지를 새로고침하거나 30초 정도 기다려주세요. 자동으로 업데이트됩니다.

### Q4. 학생 인증이 거절되었어요
A: 이메일로 발송된 거절 사유를 확인하고, 올바른 서류로 다시 제출해주세요.

### Q5. 게시글을 수정하거나 삭제할 수 있나요?
A: 네, 본인이 작성한 게시글은 수정 및 삭제가 가능합니다. 게시글 우측 상단의 메뉴를 클릭하세요.

### Q6. 목표를 변경할 수 있나요?
A: 네, 대시보드에서 언제든지 목표를 수정할 수 있습니다.

### Q7. 미니게임 점수가 제출되지 않아요
A: 인터넷 연결을 확인하고 게임을 다시 플레이해주세요. 문제가 지속되면 관리자에게 문의하세요.

### Q8. 알림을 받고 싶어요
A: 현재 이메일 알림만 지원됩니다. 브라우저 알림은 향후 업데이트 예정입니다.

## 문제 해결

### 로그인 문제

**증상**: 로그인이 되지 않음

**해결 방법**:
1. 이메일과 비밀번호를 정확히 입력했는지 확인
2. Caps Lock이 켜져있는지 확인
3. 브라우저 쿠키 및 캐시 삭제
4. 다른 브라우저로 시도

---

### 페이지 로딩 문제

**증상**: 페이지가 로딩되지 않음

**해결 방법**:
1. 인터넷 연결 확인
2. 페이지 새로고침 (F5 또는 Cmd+R)
3. 브라우저 캐시 삭제
4. 브라우저 업데이트

---

### 파일 업로드 문제

**증상**: 학생증 사진이 업로드되지 않음

**해결 방법**:
1. 파일 크기 확인 (최대 10MB)
2. 지원 형식 확인 (JPG, PNG, PDF)
3. 파일명에 특수문자가 없는지 확인
4. 다른 파일로 시도

---

### 채팅 문제

**증상**: 메시지가 전송되지 않음

**해결 방법**:
1. 인터넷 연결 확인
2. 페이지 새로고침
3. 로그아웃 후 재로그인
4. 브라우저 콘솔 에러 확인
