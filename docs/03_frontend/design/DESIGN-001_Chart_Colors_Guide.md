## 📊 색상 의미 체계

### 주요 색상 규칙 (주식 차트 기반)

| 색상 | HSL 값 | 의미 | 사용 예시 |
|------|--------|------|-----------|
| 🔴 **빨강** | `hsl(0, 70%, 75%)` | 감소, 경고, 특정 타입 | 포모도로 (25분), 하락 트렌드 |
| 🔵 **파랑** | `hsl(210, 70%, 75%)` | 안정, 기본, 중립 | 휴식 시간, 작년 데이터, 중립 |
| 🟢 **초록** | `hsl(142, 65%, 70%)` | 증가, 성공, 목표 달성 | 장기 집중, 목표 완료, 상승 |
| 🟡 **노랑** | `hsl(45, 85%, 75%)` | 주의, 진행중, 중간 | 단기 집중, 진행도 30-70% |
| 🟣 **보라** | `hsl(270, 60%, 75%)` | 특별, 업적, 프리미엄 | 플래티넘 배지, 특별 이벤트 |
| 🟠 **주황** | `hsl(25, 80%, 72%)` | 활동, 에너지, 집중 | 작업 시간, 활동량, 집중 |

## 🎯 적용된 차트별 색상

### 1. 주간 집중 시간 차트 (FocusTimeChart)
**차트 타입**: Area Chart (영역 차트)

**색상**:
- **주황색** (`hsl(25, 80%, 72%)`) - 집중 시간 (활동/에너지)
- 그라디언트: 위에서 아래로 투명도 증가

**의미**:
- 주황색 = 활동적인 집중 시간을 나타냄
- 에너지와 생산성을 시각적으로 표현

```typescript
// 적용된 코드
<linearGradient id="colorHours" x1="0" y1="0" x2="0" y2="1">
  <stop offset="5%" stopColor="hsl(25, 80%, 72%)" stopOpacity={0.4} />
  <stop offset="95%" stopColor="hsl(25, 80%, 72%)" stopOpacity={0} />
</linearGradient>
```

---

### 2. 세션 분포 차트 (SessionDistributionChart)
**차트 타입**: Pie Chart (파이 차트)

**색상 매핑**:
| 세션 타입 | 색상 | HSL 값 | 의미 |
|-----------|------|--------|------|
| 포모도로 (25분) | 🔴 빨강 | `hsl(0, 70%, 75%)` | 표준 포모도로 기법 |
| 단기 집중 (15분) | 🟡 노랑 | `hsl(45, 85%, 75%)` | 짧은 집중 (주의) |
| 장기 집중 (50분+) | 🟢 초록 | `hsl(142, 65%, 70%)` | 긴 집중 (성공) |
| 휴식 | 🔵 파랑 | `hsl(210, 70%, 75%)` | 안정/회복 |

**의미**:
- 빨강 = 포모도로 기법의 표준 시간
- 노랑 = 짧은 시간 (주의 필요)
- 초록 = 긴 시간 (성공적인 집중)
- 파랑 = 휴식 (안정)

```typescript
// 적용된 코드
const sessionDistribution = [
  { name: "포모도로 (25분)", value: X, color: "hsl(0, 70%, 75%)" },
  { name: "단기 집중 (15분)", value: Y, color: "hsl(45, 85%, 75%)" },
  { name: "장기 집중 (50분)", value: Z, color: "hsl(142, 65%, 70%)" },
  { name: "휴식", value: W, color: "hsl(210, 70%, 75%)" },
];
```

### 3. 주간 활동 차트 (WeeklyActivityChart)
**차트 타입**: Bar Chart (막대 그래프)

**색상**:
- **주황색** (`hsl(25, 80%, 72%)`) - 기본 막대
- **진한 주황** (`hsl(25, 80%, 65%)`) - 호버 상태

**의미**:
- 주황색 = 활동적인 집중 시간
- 호버 시 진해짐 = 상호작용 피드백

```typescript
// 적용된 코드
fill={
  activeIndex === index
    ? "hsl(25, 80%, 65%)" // 진한 오렌지 (hover)
    : "hsl(25, 80%, 72%)" // 밝은 파스텔 오렌지
}
```

---

### 4. 월별 비교 차트 (MonthlyComparisonChart)
**차트 타입**: Bar Chart (막대 그래프)

**색상** (권장):
- **초록색** (`hsl(142, 65%, 70%)`) - 올해 데이터 (현재/성장)
- **파랑색** (`hsl(210, 70%, 75%)`) - 작년 데이터 (과거/기준)

**의미**:
- 초록 = 올해 (성장/진행중)
- 파랑 = 작년 (기준/비교 대상)

### 5. 시간대별 패턴 차트 (HourlyPatternChart)
**차트 타입**: Line Chart (라인 차트)

**색상** (권장):
- **주황색** (`hsl(25, 80%, 72%)`) - 세션 수
- **초록색** (`hsl(142, 65%, 70%)`) - 평균 시간

**의미**:
- 주황 = 활동량 (세션 수)
- 초록 = 성과 (평균 시간)

## 📋 색상 시스템 파일

### `src/utils/chart-colors.ts`

전체 색상 시스템을 정의한 유틸리티 파일:

```typescript
export const chartColors = {
  // 주요 색상 (파스텔 톤)
  primary: {
    blue: 'hsl(210, 70%, 75%)',
    green: 'hsl(142, 65%, 70%)',
    red: 'hsl(0, 70%, 75%)',
    yellow: 'hsl(45, 85%, 75%)',
    purple: 'hsl(270, 60%, 75%)',
    orange: 'hsl(25, 80%, 72%)',
  },

  // 세션 타입별 색상
  session: { ... },

  // 트렌드 색상
  trend: {
    up: 'hsl(142, 65%, 70%)',    // 그린
    down: 'hsl(0, 70%, 75%)',     // 레드
    neutral: 'hsl(210, 70%, 75%)', // 블루
  },

  // 목표 진행도 색상
  progress: {
    low: 'hsl(0, 70%, 75%)',      // 0-30%
    medium: 'hsl(45, 85%, 75%)',  // 31-70%
    high: 'hsl(25, 80%, 72%)',    // 71-99%
    complete: 'hsl(142, 65%, 70%)', // 100%
  },
};
```

**헬퍼 함수**:
- `getProgressColor(percentage)` - 진행도에 따른 색상
- `getIntensityColor(hours)` - 활동 강도에 따른 색상
- `getTrendColor(value, previousValue)` - 트렌드에 따른 색상

## 🎯 사용 가이드

### 새로운 차트 추가 시

```typescript
import { chartColors } from '@/utils/chart-colors';

// 1. 세션 타입별 색상
<Bar fill={chartColors.session.work} />

// 2. 트렌드 색상
<Line stroke={chartColors.trend.up} />

// 3. 진행도 색상
import { getProgressColor } from '@/utils/chart-colors';
<Circle fill={getProgressColor(75)} />
```

### 색상 선택 기준

| 데이터 유형 | 추천 색상 | 이유 |
|------------|----------|------|
| 집중/작업 시간 | 주황 | 활동, 에너지 |
| 휴식 시간 | 파랑 | 안정, 회복 |
| 목표 달성 | 초록 | 성공, 완료 |
| 경고/주의 | 노랑 | 주의 필요 |
| 감소/하락 | 빨강 | 경고, 감소 |
| 특별 이벤트 | 보라 | 프리미엄 |
