## 🔴 P0: 긴급 수정 사항

### 1. 방 생성/참여 플로우 버그
**문제**: 방 생성 후 "내 방" 목록에 표시 안 됨

**영향**: 사용자가 생성한 방을 다시 찾을 수 없음 (Critical UX Issue)

**해결 방법**:
- `routes/index.tsx`: 방 생성 후 자동 참여 로직 추가
- `MyRoomsSection.tsx`: API 연동 완료 (이미 구현됨)
- 방 나가기 후 상태 동기화

### 2. 메시지 송수신 미구현
**문제**: WebSocket 연결은 되지만 실제 메시지 전송 안 됨

**영향**: 채팅 기능 사용 불가

**해결 방법**:
- `Messages.tsx`: 메시지 전송 API 연동
- WebSocket 메시지 핸들러 구현
- 낙관적 UI 업데이트

### 3. 예약 날짜/시간 입력 에러
**문제**: "Malformed value" 에러 발생

**영향**: 예약 기능 사용 불가

**해결 방법**:
```typescript
// ISO 8601 형식으로 변환
const formattedDateTime = `${date}T${time}:00`;
```
