# FocusMate 이미지 요구사항 명세서

> **분석 범위**: 전체 17개 페이지 + 컴포넌트

## 📊 전체 요약

| 카테고리 | 필요 이미지 수 | 우선순위 |
|---------|--------------|---------|
| 로고 & 브랜딩 | 5개 | 🔴 높음 |
| Empty State 일러스트 | 12개 | 🔴 높음 |
| 기능 설명 일러스트 | 6개 | 🟡 중간 |

## 1️⃣ 로고 & 브랜딩 (5개)

### 1.1 메인 로고 아이콘
- **위치**:
  - `Home.tsx` (L48-54): 헤더 로고 원형 컨테이너
  - `Auth.tsx`: 로그인/회원가입 페이지 상단
  - 모든 페이지 네비게이션 바
- **크기**: 48x48px (w-12 h-12)
- **현재 상태**: ⚠️ Lucide Timer 아이콘 사용 중
- **컨셉**:
  - 타이머/시계 + 협업 요소 결합
  - 블루-퍼플 그라디언트 (#3B82F6 → #8B5CF6)
  - 원형 배경에 흰색 아이콘
  - 미니멀하고 현대적인 디자인

### 1.2 전체 로고 (아이콘 + 텍스트)
- **위치**:
  - 마케팅 페이지
  - 이메일 템플릿
  - 문서 헤더
- **크기**: 200x60px
- **컨셉**:
  - 아이콘 + "Focus Mate" 텍스트
  - 산세리프 폰트 (Inter, SF Pro)
  - 그라디언트 또는 단색

### 1.3 Favicon
- **위치**: 브라우저 탭 (`index.html`)
- **크기**: 32x32px, 16x16px
- **현재 상태**: ❌ 기본 favicon
- **컨셉**:
  - 로고 아이콘의 단순화 버전
  - 작은 크기에서도 인식 가능
  - ICO 형식

### 1.4 다크모드 로고
- **위치**: 다크모드 활성화 시 모든 위치
- **크기**: 로고와 동일
- **컨셉**:
  - 밝은 배경용 버전
  - 흰색/밝은 색상 사용

### 1.5 앱 아이콘 (PWA)
- **위치**: PWA 설치 시
- **크기**: 512x512px, 192x192px
- **컨셉**:
  - 메인 로고와 동일한 디자인
  - 고해상도 버전

  - 블루-퍼플 그라디언트

### 2.10 빈 Hall of Fame

- **컨셉**:

## 3️⃣ 기능 설명 일러스트 (6개)

- **크기**: 300x300px
- **현재 상태**: ⚠️ 숫자 "1"만 표시
- **컨셉**:
  - 사람이 "방 만들기" 버튼 클릭
  - 문이 열리는 메타포
  - 친근하고 현대적인 플랫 디자인

### 3.2 Step 2 - 함께 집중
- **위치**: `Home.tsx` (L132-143)
- **크기**: 300x300px
- **현재 상태**: ⚠️ 숫자 "2"만 표시
- **컨셉**:
  - 여러 사람 + 동기화된 타이머
  - 집중하는 모습
  - 협업적이고 생산적인 분위기

### 3.3 Step 3 - 함께 휴식
- **위치**: `Home.tsx` (L146-157)
- **크기**: 300x300px
- **현재 상태**: ⚠️ 숫자 "3"만 표시
- **컨셉**:
  - 사람들이 휴식 중
  - 커피 컵, 평화로운 분위기
  - 차분한 색상

### 3.4 포모도로 타이머 설명
- **위치**: 온보딩/튜토리얼
- **크기**: 400x300px
- **컨셉**:
  - 타이머 + 집중/휴식 사이클
  - 인포그래픽 스타일
  - 교육적이면서 시각적

### 3.5 매칭 시스템 설명
- **위치**: `Matching.tsx` - 도움말 섹션
- **크기**: 400x300px
- **컨셉**:
  - 두 그룹이 매칭되는 과정
  - 하트 + 화살표
  - 핑크-퍼플 톤

### 3.6 랭킹 시스템 설명
- **위치**: `Ranking.tsx` - 도움말 섹션
- **크기**: 400x300px
- **컨셉**:
  - 팀들이 경쟁하는 모습
  - 트로피 + 차트
  - 동기부여하는 분위기

## 4️⃣ 배경 & 장식 (8개)

### 4.1 히어로 배경 (Home)
- **위치**: `Home.tsx` (L25) - 전체 배경
- **크기**: 1920x1080px (반복 가능)
- **현재 상태**: ⚠️ 단색 그라디언트 (`bg-muted/30`)
- **컨셉**:
  - 부드러운 블루-퍼플 그라디언트
  - 미묘한 기하학적 패턴
  - 전문적이고 차분한 분위기

### 4.2 매칭 페이지 배경
- **위치**: `Matching.tsx` (L83) - 전체 배경
- **크기**: 1920x1080px
- **현재 상태**: ✅ 그라디언트 (`from-pink-50 via-purple-50 to-blue-50`)
- **컨셉**:
  - 핑크-퍼플-블루 그라디언트
  - 로맨틱하지만 세련된
  - 부드러운 질감

### 4.3 프로필 배경
- **위치**: `Profile.tsx` - 상단 헤더
- **크기**: 1200x300px
- **컨셉**:
  - 추상적인 그라디언트
  - 블루-퍼플 톤
  - 전문적이고 차분한

### 4.4 Hall of Fame 배경
- **위치**: `HallOfFame.tsx` - 상단 배너
- **크기**: 1200x200px
- **컨셉**:
  - 트로피, 별, 업적 테마
  - 골드/실버 악센트
  - 동기부여하는 분위기

### 4.5 패턴 타일 (전역)
- **위치**: 다양한 섹션 배경
- **크기**: 200x200px (타일 가능)
- **컨셉**:
  - 미묘한 기하학적 패턴
  - 밝은 회색/흰색 톤
  - 반복 가능, 방해되지 않음

### 4.6 카드 배경 패턴
- **위치**: 주요 카드 컴포넌트
- **크기**: 600x400px
- **컨셉**:
  - 부드러운 그라디언트
  - 미묘한 노이즈/질감
  - 깊이감 있는 디자인

### 4.7 404 에러 페이지 배경
- **위치**: 404 페이지
- **크기**: 1920x1080px
- **컨셉**:
  - 유머러스하지만 전문적
  - 길을 잃은 사람 + 지도
  - 밝고 친근한 색상

### 4.8 로딩 배경
- **위치**: 로딩 스크린
- **크기**: 1920x1080px
- **컨셉**:
  - 애니메이션 가능한 그라디언트
  - 차분하고 현대적
  - 블루-퍼플 톤

- **컨셉**:
  - 1위: 골드 트로피
  - 2위: 실버 메달
  - 3위: 브론즈 메달
